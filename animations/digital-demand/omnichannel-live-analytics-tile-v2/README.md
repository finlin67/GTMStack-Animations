# Omnichannel Live Analytics Tile

An omnichannel live analytics tile that presents channel performance, conversion behavior, and revenue impact in one view.

## Description
The Omnichannel Live Analytics Tile is a high-fidelity, standalone dashboard designed to provide a real-time bird's-eye view of business performance across multiple sales channels. This compact 600x600px interface leverages glassmorphism aesthetics and vibrant neon accents to deliver critical data visualizations with clarity and style. It features a dynamic conversion funnel, live activity feeds, and essential KPIs like total sales, ROI, and active traffic. The application uses simulated real-time data to demonstrate its capability for rapid situational awareness and cross-platform monitoring.

## Tech Stack
- **Framework:** React 19
- **Styling:** Tailwind CSS (Dark Mode, Custom Glassmorphism)
- **Animations:** Framer Motion (Layout transitions, Presence-based lists)
- **Icons:** Lucide React
- **Module System:** ES6 Modules with ESM.sh imports

## Usage
1. **View Dashboard:** The application automatically starts on the main dashboard view, displaying live stats that update every few seconds.
2. **Monitor Funnel:** Observe the multi-stage conversion funnel to identify where potential customers are dropping off in the journey.
3. **Track Activity:** The "Live Activity" section provides a scrolling feed of recent user events, such as purchases or channel switches.
4. **Export & Configure:** Use the bottom actions to simulate exporting reports or accessing view configuration settings.
5. **Responsive Design:** The tile is strictly constrained to a 600x600px square, making it ideal for integration into larger modular control rooms or as a standalone monitoring tile.
