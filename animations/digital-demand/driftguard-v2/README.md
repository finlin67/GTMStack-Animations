# DriftGuard - AI Performance & Drift Alerts

A compact monitoring tile for AI model accuracy, drift signals, and real-time alerting.

## Description

DriftGuard is a compact, real-time dashboard designed for monitoring the performance and health of AI models. It provides an at-a-glance view of critical metrics, including model accuracy, prediction drift, data drift, and concept drift, presented within a sleek 600x600px interface. The dashboard features a futuristic, cyber-themed design with glassmorphism panels and dynamic animations to visualize data jitter and live events. A real-time alert feed instantly notifies operators of significant events, such as high-confidence drift detection or model deployment updates, allowing for rapid response to maintain model integrity.

## App Name

DriftGuard

## Tech Stack

*   **React 19**: For building the user interface with modern features.
*   **TypeScript**: For static typing and improved code quality.
*   **Tailwind CSS**: For a utility-first styling workflow.
*   **Framer Motion**: For fluid animations and interactive elements.
*   **Lucide React**: For a lightweight and consistent icon set.

## Usage

This project is a standalone component bundled into a single `index.html` file with no external build dependencies.

To view the application, simply open the `index.html` file in any modern web browser.
