# EdTech Compact ROI Funnel

A compact ROI funnel visualization tailored for education technology growth and conversion tracking.

## Description

This tile presents the key stages of an education funnel in a clean, high-contrast layout, with animated counters and progress indicators for conversion metrics. It is designed to fit a 600x600 hero tile while preserving funnel clarity.

## Tech Stack

- React
- Tailwind CSS
- Framer Motion
