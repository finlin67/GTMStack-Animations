# PowerGrid Digital Modernization Dashboard

A PowerGrid modernization tile built to present capital progress, reliability uplift, and lifecycle execution signals.

### Key Features
- **Hero Tile Optimization**: Designed specifically for 600x600 square viewports with adaptive font scaling and grid refactoring.
- **Real-time Data Jitter**: Uses a recursive state update logic to simulate live data streams for "Managed Capital" and "Grid Reliability".
- **Interactive Lifecycle Spoke**: A central visualization hub with floating status nodes that represent different stages of grid modernization.
- **Theme Support**: Seamless transitions between Light and Industrial Dark modes.
- **Progress Tracking**: Dynamic 36-month cycle progression bar with phase-based highlighting.

### Technical Stack
- **React**: Core UI logic and component lifecycle.
- **Framer Motion**: Handles energy line glows, floating node animations, and central hub pulse effects.
- **Tailwind CSS**: Custom utility classes for grid backgrounds and industrial "energy-line" gradients.
- **Lucide React**: Vector-perfect industrial icons.
