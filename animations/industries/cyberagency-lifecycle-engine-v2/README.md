# CyberAgency Lifecycle Engine

A cybersecurity lifecycle tile that maps audit-to-deployment progress with live risk and remediation visibility.

## Tech Stack
- **Framework:** React 19 (Functional Components & Hooks)
- **Styling:** Tailwind CSS (Modern Utility-First UI)
- **Animations:** Framer Motion (Smooth Transitions & High-Fidelity Physics)
- **Icons:** Lucide React (Clean, Accessible Vector Icons)
- **Design Pattern:** High-Fidelity Surface approach with theme randomization (Professional Light & Modern Deep modes).

## Usage
1. **View Engine:** On load, the app randomly selects between a Professional Light (Warm Vanilla) or Modern Deep (Navy/Slate) theme.
2. **Track Progress:** Observe the central Lifecycle Timeline showing progress through Audit, Hardening, Approval, and Deployment phases.
3. **Analyze Data:** Monitor "Threat Dynamics" and "Node Resilience" widgets for live jitter-based data simulation.
4. **Execute Audit:** Click the primary "Initiate Lifecycle Audit" button to trigger a system-wide diagnostic scan, which updates the security posture in real-time.
