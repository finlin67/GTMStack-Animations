
# Enterprise Sales Motion Dashboard

A premium SaaS sales lifecycle management interface designed for high-touch enterprise deal tracking. This dashboard provides a dynamic, interactive visualization of the sales journey from discovery to implementation.

This project is a single, standalone React component named `EnterpriseSalesMotionDashboard`.

## Key Features

- **Interactive Timeline Scale**: Adjust the sales cycle duration (3, 6, or 9 months) and watch the UI adapt its complexity and phase indicators in real-time.
- **Dynamic Stakeholder Mapping**: Visual representation of Champions and Buyers with interactive multi-threading details.
- **Real-Time Sales Metrics**: Live-simulated metrics for stakeholder engagement, average cycle days, and motion efficiency using organic jitter logic.
- **Champion Profile Insights**: Hover over the central "Champion" node to reveal deep-dive influence and engagement analytics via an overlaid insights panel.
- **Responsive Hero Tile Design**: Fully optimized aspect-square layout that scales fluidly for mobile while maintaining a clean 600px constraint on desktop.
- **Advanced Animations**: Powered by Framer Motion for smooth transitions, pulsed connection lines, and focus-aware UI overlays.

## Technology Stack

- **React 19**: Modern component architecture.
- **Tailwind CSS**: Utility-first styling with a custom dark-mode slate theme.
- **Framer Motion**: Smooth, declarative animations and state transitions.
- **Lucide React**: Clean, consistent enterprise-grade iconography.

## Usage

Interact with the duration slider at the bottom to see how the enterprise motion complexity scales over time. Hover over the "Zap" icon in the center hub to view specific Champion metrics in the Selection phase card.
