# Live Email Automation Ecosystem

A live email automation tile that presents campaign flow, delivery behavior, and lifecycle performance in a compact operations view.

## Features
- **Real-time Stats Engine**: Organic jittering revenue and performance metrics for a "live" feel.
- **Animated Workflow**: Step-by-step visual flow with comet-particle animations and glow effects.
- **Responsive Scaling**: Optimized to fit within a 600x600 container (Hero Tile) while remaining mobile-responsive.
- **Glassmorphism Design**: Premium aesthetics using backdrop blurs, subtle borders, and deep gradients.

## Usage
Import the `LiveEmailAutomation` component into any React project. Ensure `framer-motion`, `lucide-react`, and `tailwindcss` are installed.

```tsx
import LiveEmailAutomation from './Live-Email-Automation';

function Dashboard() {
  return <LiveEmailAutomation />;
}
```
