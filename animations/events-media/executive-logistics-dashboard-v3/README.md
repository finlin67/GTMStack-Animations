# Executive Logistics Dashboard

A high-fidelity, interactive logistics dashboard featuring real-time node simulation, advanced route filtering, and persistent data visualization using a 'Modern Deep' surface design.

### Tech Stack
* **React 19** for UI and state management.
* **Tailwind CSS** for strictly constrained (600x600px) responsive styling.
* **Framer Motion** for high-performance interactive animations.
* **Lucide React** for consistent iconography.

### Usage
This component is designed as a standalone hero tile. Click on any global node on the map to focus its specific routes and view detailed throughput analytics. Click the background to deselect. Data jitters organically to simulate live sensor feedback.
