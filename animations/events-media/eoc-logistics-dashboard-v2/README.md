# EOC Logistics Dashboard

An EOC logistics dashboard tile designed to track incident flow, route stability, and emergency operations performance.

## Features
- **Real-time Data Simulation**: Tracks deviation, incidents, efficiency, and latency with organic jitter.
- **Brutalist Design**: High contrast, raw aesthetic using pure Tailwind CSS.
- **Responsive Animations**: Framer Motion integration for alerting and scanning effects.
- **Adaptive Layout**: Scales gracefully from mobile to desktop (max 600px).

## Tech Stack
- React 19
- Tailwind CSS
- Framer Motion
- Lucide React

## Usage
Import the default export from `App.tsx` to use the dashboard component. The component handles its own responsive sizing within a full-width container.
