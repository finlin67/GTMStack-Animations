# Marketing Automation Workflow

A biomimetic workflow visualization that maps nurture paths and routing decisions in motion.

## Context & Creative Strategy

This tile makes marketing automation tangible by rendering lead movement as organic particle flows. The design emphasizes clarity and momentum to help stakeholders understand routing logic and performance impact.

## Purpose & Value Proposition

- Visual clarity for nurture vs. re-engage decisions.
- Real-time feedback on engagement probability and lead volume.

## Tech Stack

- React
- Tailwind CSS
- Framer Motion
