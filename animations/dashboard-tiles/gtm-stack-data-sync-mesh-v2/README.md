# GTM Stack Data Sync Mesh

A RevOps command-center tile visualizing data sync health across core GTM systems.

## Context & Creative Strategy

This component uses a hub-and-spoke metaphor to show data flow between tools like Salesforce, HubSpot, and billing systems. Animated status indicators surface integration health and sync latency at a glance.

## Tech Stack

- React
- Tailwind CSS
- Framer Motion
